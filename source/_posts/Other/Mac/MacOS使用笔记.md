## 外置键盘键位调整

默认是alt对应Option，win键对应Meta，跟Mac的键位不一样。可以改成alt对应Meta，win对应Option：

系统设置 -> 键盘 -> 键盘快捷键 -> 修饰键，把`Option键`设置成`Command键`，把`Command键`设置成`Option键`。

## 禁止自动重新排列空间

系统设置 -> 桌面与程序坞 -> 调度中心 -> 关闭`根据最近使用情况自动重新排列空间`

## 外置显示器字小

MacOS不支持调整外置显示器上的缩放比，所以好像只能把分辨率调低了。

## 设置开机自启应用

系统设置 -> 通用 -> 登录时打开 -> 加号 -> 应用程序，选择要开机自启的应用，点击`打开`即可。

## 快捷键

系统设置 -> 键盘 -> 键盘快捷键。这里放一些个人觉得比较有用的快捷键。

调度中心 -> 调度中心 -> 切换到桌面1/2/3/...

## 第三方应用

### AltTab

```shell
brew install alt-tab
```

打开AltTab，授予权限，然后打开app的设置，在里面把快捷键2从`` Option + ` `` 改成 `` Command + ` ``。会提示快捷键冲突，要到系统设置 -> 键盘 -> 键盘快捷键 -> 键盘 里取消勾选`将焦点移到下一个窗口`。
