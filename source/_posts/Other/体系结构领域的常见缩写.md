英文缩写简洁，但是不易懂……

| 缩写 | 全称 | 中文含义 |
| ---- | ---- | ----|
| I$ | Instruction Cache | 指令缓存 |
| RF | Register File | 寄存器堆 |
| LSU | Load Store Unit| 存取单元 |
| IF | Instruction Fetch | 取指 |
| ID | Instruction Decode | 解码 |
| EX | execute | 执行 |
| WB | write-back | 写回 |
